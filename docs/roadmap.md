# Roadmap

## v1.0 (Current)

- [x] Finder context menu integration
- [x] Git URL validation (HTTPS, SSH, all providers)
- [x] Clone with progress tracking
- [x] Smart project detection (Xcode, VS Code, Cursor, IntelliJ, etc.)
- [x] Auto-open in detected editor
- [x] Cancellation support
- [x] Error handling with suggestions
- [x] Preferences (default editor, auto-open)
- [x] Homebrew Cask installation

## v1.1 (Planned)

- [ ] Clone specific branches
- [ ] Shallow clone option
- [ ] Git LFS support
- [ ] Clone history tracking
- [ ] Quick Clone (recent repositories)

## v1.2 (Planned)

- [ ] Post-clone actions (install dependencies, copy .env, etc.)
- [ ] Custom post-clone scripts
- [ ] Dev Container detection and setup
- [ ] Docker Compose auto-start

## v2.0 (Future)

- [ ] Multi-editor support with per-project overrides
- [ ] Repository templates
- [ ] Organization-specific clone workflows
- [ ] Git provider API integration (star, fork, etc.)
- [ ] Notification system for clone completion
- [ ] Spotlight integration
- [ ] Shortcuts integration
